# OpenVINO Electron App

## Usage

Select task and click "Open Example"

## Development

Run dev
`npm run dev`

Build
`npm run build`

## Algorithm

### Home Screen

- Select task

### Task Screen

- get model list for task
  - get predefined models
  - get loaded models
- select default model

- get input type (task)
- preprocess input

- perform inference

- postprocess input

- show result
